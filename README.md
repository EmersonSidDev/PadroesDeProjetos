# 📘 Padrões de Projetos de Software em Java

Este repositório foi criado como parte da disciplina **Padrões de Projetos de Software**, com o objetivo de organizar e demonstrar, de forma prática, a implementação dos principais padrões de projeto utilizando a linguagem **Java**.

## 📂 Estrutura do Repositório

A organização do repositório é baseada em **branches**, onde **cada branch representa um padrão de projeto ou conteúdo específico da disciplina**.

Isso permite uma separação clara dos exemplos, facilitando o estudo individual de cada padrão.

### 🔀 Branches atuais

Atualmente, o repositório contém as seguintes branches:

* `simple-factory` → Exemplo do padrão *Simple Factory*
* `factory-method` → Implementação do padrão *Factory Method*
* `abstract-factory` → Implementação do padrão *Abstract Factory*
* `singleton` → Implementação do padrão *Singleton*
* `builder` → Implementação do padrão *Builder*

## 🚀 Objetivo

Este projeto tem como principais objetivos:

* Consolidar o aprendizado dos padrões de projeto
* Servir como material de apoio para estudo
* Demonstrar exemplos práticos em Java
* Facilitar futuras consultas e revisões

## 📈 Expansão futura

Novas branches serão adicionadas ao longo da disciplina, incluindo outros padrões e conteúdos, como:

* Padrões estruturais
* Padrões comportamentais
* Boas práticas de design
* Exemplos adicionais e variações

## 🛠️ Como utilizar

1. Clone o repositório:

   ```bash
   git clone https://github.com/EmersonSidDev/PadroesDeProjetos.git
   ```

2. Acesse a branch desejada:

   ```bash
   git checkout nome-da-branch
   ```

3. Explore o código e os exemplos disponíveis

## 📚 Observações

* Cada branch é independente e contém seu próprio contexto e implementação
* Recomenda-se estudar um padrão por vez
* Os exemplos podem evoluir com melhorias ao longo do tempo
